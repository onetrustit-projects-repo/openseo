# AI-Powered Content Optimization

Semantic content optimization with AI-powered suggestions.

## Features

- **Content Analysis**: NLP-based content scoring
- **Content Brief Generator**: AI-generated briefs from keywords
- **Meta Description Generator**: Auto-generated with character limits
- **Readability Suggestions**: Sentence length, passive voice
- **Topic Suggestions**: Related topics based on semantic analysis
- **Heading Structure**: H1-H6 optimization tips

## API Endpoints

- `POST /api/content/analyze` - NLP content analysis
- `POST /api/content/score` - Overall content score
- `POST /api/content/meta-description` - Generate meta descriptions
- `POST /api/brief/generate` - Generate content brief
- `POST /api/suggestions/readability` - Readability tips
- `POST /api/suggestions/topics` - Related topics
- `POST /api/suggestions/headings` - Heading suggestions
- `POST /api/suggestions/semantics` - Semantic enhancements

## Quick Start

```bash
cd api && npm install && npm run dev
cd frontend && npm install && npm run dev
```
